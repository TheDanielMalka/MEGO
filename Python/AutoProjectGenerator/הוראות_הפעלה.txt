הפרויקט נוצר אוטומטית בתיקייה:
c:\Users\shimo\MEGO\Python\Functions\Projects


═══════════════════════════════════════════════════════════════
   דרך 1: הרצה ידנית (פשוט)
═══════════════════════════════════════════════════════════════

1. פתח CMD או Terminal

2. הרץ את הפקודה:
   python "c:\Users\shimo\MEGO\Python\AutoProjectGenerator\daily_project_generator.py"

3. זהו! נוצר פרויקט חדש


═══════════════════════════════════════════════════════════════
   דרך 2: הפעלה אוטומטית עם Windows Task Scheduler
═══════════════════════════════════════════════════════════════

שלב 1: פתיחת Task Scheduler
------------------------------
1. לחץ על כפתור Windows + R
2. הקלד: taskschd.msc
3. לחץ ENTER


שלב 2: יצירת משימה חדשה
-------------------------
1. לחץ על "Create Basic Task" בצד ימין
2. שם: "Daily Project Generator"
3. תיאור: "יוצר פרויקט פייתון חדש כל יום"
4. לחץ Next


שלב 3: קביעת תזמון
------------------
1. בחר: "Daily" (יומי)
2. לחץ Next
3. בחר את השעה שאתה רוצה (למשל 20:00)
4. לחץ Next


שלב 4: הגדרת הפעולה
-------------------
1. בחר: "Start a program"
2. לחץ Next
3. בשדה "Program/script" הדבק:
   python

4. בשדה "Add arguments" הדבק:
   "c:\Users\shimo\MEGO\Python\AutoProjectGenerator\daily_project_generator.py"

5. לחץ Next


שלב 5: סיום
-----------
1. סמן את התיבה "Open the Properties dialog..."
2. לחץ Finish
3. בחלון שנפתח:
   - לחץ על הטאב "Conditions"
   - בטל את הסימון של "Start the task only if the computer is on AC power"
4. לחץ OK


═══════════════════════════════════════════════════════════════
   דרך 3: הרצה עם קובץ BAT (הכי פשוט)
═══════════════════════════════════════════════════════════════

יצרתי לך קובץ run_generator.bat

אפשר גם ליצור קיצור דרך שלו על שולחן העבודה:
1. לחץ ימני על run_generator.bat
2. Send to -> Desktop (create shortcut)
