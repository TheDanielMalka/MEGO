כל משחק חדש שנוצר יכיל:

1. import של game_auth מהתיקייה GameData
2. GAME_NAME = "שם המשחק"
3. פונקציה play_game_round(username) - הלוגיקה של המשחק
4. פונקציה game_menu(username) - תפריט המשחק המלא
5. פונקציה main() - התחברות והפעלת התפריט

כל הנתונים נשמרים אוטומטית:
- כל משחק מקבל קובץ משתמשים משלו: {game_name}_users.json
- כל המשחקים משתפים קובץ ליידרבורד אחד: all_games_leaderboard.json

הקבצים מתעדכנים אוטומטית בזמן המשחק!
