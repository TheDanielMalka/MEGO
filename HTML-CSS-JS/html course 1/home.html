How to download VS Code?

It's very simple!
So let's do it:
  1. Go to VS Code download page:
    click here
  2. Click on the button under your operating system (probably windows)
  3. Open the file after the download and install the prograp
  4. Open VS Code, look around
    When you're ready, hit CTRL+K to open a folder
    You should then see the folder's files in the Explorer section
    You can create new files and folders in VS Code
